## 创建和使用类
在 Python 中，<font color="#245bdb">⾸字⺟⼤写的名称指的是类</font>
```python
class Dog: 
	"""⼀次模拟⼩狗的简单尝试""" 
	def __init__(self, name, age):
	 """初始化属性 name 和 age""" 
		self.name = name 
		self.age = age 
		
	def sit(self):
		 """模拟⼩狗收到命令时坐下"""
		 print(f"{self.name} is now sitting.")
		 
	def roll_over(self): 
		"""模拟⼩狗收到命令时打滚""" 
		print(f"{self.name} rolled over!")
```
