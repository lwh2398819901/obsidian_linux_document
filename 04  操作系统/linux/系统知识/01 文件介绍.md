## 文件类型
linux一共有7种文件类型,分别如下:

| 文件类型 |            解释           |
| :--: | :--------------------- |
|   -  |           普通文件          |
|   d  |           目录文件          |
|   l  |   软链接（类似Windows的快捷方式）   |
|   b  |  块设备文件（例如硬盘、光驱等可随机存取装置） |
|   p  |           管道文件          |
|   c  | 字符设备文件（例如键盘　鼠标等一次性读取装置） |
|   s  |       套接口文件/数据接口文件      |
